What does "Full" in Full Stack Mean

Full means that you develop both the front end, and the back end. The front end is what the user sees on what is called the client computer. The back end is what is on the server computer that delivers content to the front end.


What does "stack" in full stack mean?

Stack Means that several Programs, a "stack" of programs is used. On the front end I use HTML 5, CSS3, and JavaScript. On the back end I use PHP, and MySQL.

What does "LAMP" stack mean?

Lamp Stands for:
L = Linux, which is the basis of our server operating system
A = Apache, which is our web server
M = MySql, the database we use
P = PHP, which is the programming laungage we use on the server side
